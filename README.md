# TauCodes' Pizzeria
## Frontend - React + TypeScript + Vite + Tailwind + Axios

This is the **frontend repository** for the full-stack application. It is built using **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **Axios** for seamless API integration.
